## Short Description

MagicBits helps designers quickly generate lively moodboards, sticker sheets, and visual exploration boards from selected layers or components.

## Full Description

MagicBits is a Figma and FigJam plugin for creating ScatterBoards: fast, playful arrangements generated from the layers you already selected.

Select one or more components, frames, groups, text layers, vectors, or shapes, then choose a layout style and generate a new board. MagicBits can create random clouds, grids, golden spirals, concentric rings, masonry layouts, hex grids, starbursts, organic distributions, clusters, zigzags, and vortex-style compositions.

You can tune copy count, scale range, rotation, board size, grid columns, jitter, overlap spacing, and layer order. Once a ScatterBoard is created, select it again to reroll the existing contents without making new copies.

The plugin UI supports English and Chinese. It follows the user's browser language by default and includes an in-panel language toggle.

Great for:

- Moodboards and visual research boards
- Sticker sheets and asset exploration
- Component variety checks
- Fast composition experiments

MagicBits works locally inside your Figma file and does not request network access.
