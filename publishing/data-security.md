# Data Security Disclosure Draft

Use this as the basis for Figma's optional Data security page during publishing.

## Does MagicBits access the network?

No. The manifest declares `networkAccess.allowedDomains` as `["none"]`.

## Does MagicBits collect personal data?

No.

## Does MagicBits send file content to a third party?

No.

## Does MagicBits store data?

Only inside the user's Figma file, as Figma plugin data attached to generated ScatterBoard frames. This stores layout settings so users can reroll a board later. It is not transmitted by MagicBits.

## Does MagicBits require an account, payment, license key, or external service?

No.

## Does MagicBits use AI or external generation services?

No.
