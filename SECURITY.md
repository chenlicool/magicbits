# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.0.x   | ✅ Active support   |

## Reporting a Vulnerability

MagicBits is a fully offline Figma plugin. Because it has no network access, the attack surface is limited to the Figma plugin sandbox environment.

If you discover a security vulnerability, please **do not** open a public issue. Instead, send a report to:

**chenlicool@foxmail.com**

Please include the following details:

- A brief description of the vulnerability
- Steps to reproduce
- Figma version and OS where it was observed
- Any relevant screenshots or code snippets

### Response timeline

- **Within 48 hours**: Acknowledgment of receipt
- **Within 7 days**: Initial assessment and expected fix timeline
- **Within 30 days**: A fix will be released or a mitigation plan communicated

We appreciate responsible disclosure and will acknowledge your contribution in release notes if you wish.
